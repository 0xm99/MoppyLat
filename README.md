## Install python pip
> apt install python3-pip -y

## Install web3
> pip install web3

>{
>
>    "endpoints": ["https://bsc-dataseed1.binance.org/","http://127.0.0.1:8000","ws://127.0.0.1:8546"]

> python3 MoppyLat.py
